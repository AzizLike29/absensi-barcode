# WinniPresence — Sistem Absensi Karyawan

Wireframe UI sistem absensi karyawan PT Winnicode Garuda Teknologi.
Dibuat dengan HTML, Tailwind CSS, dan Alpine.js.

## Menjalankan

Buka `index.html` di browser, atau pakai Live Server di VS Code.

## Halaman

Karyawan: dashboard, absensi (face recognition), riwayat, cuti & izin, profil.
HR Admin: dashboard, data karyawan, titik lokasi, rekap & laporan.

Login: pilih peran. Karyawan lewat verifikasi OTP (kode demo: `123456`).
